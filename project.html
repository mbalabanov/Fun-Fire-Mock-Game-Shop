About this Project

Video Introduction to Fun Fire1

This is an example project as part of the courses Full Stack Web Developer and Frontend Developer at the Code Factory. The Fun Fire games shop was created in four cold days in February 2020 by Mireille Bebon, Yana Nikolaienko, Sepide Nickmanesh and Marin Balabanov. We proudly call ourselves the Dragon Team. Find out more about the team members on the team page.
